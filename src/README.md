# src/

Esta carpeta se estructura en el **Paso 5 (Arquitectura)**, según el stack elegido.

Ejemplos de cómo podría verse, dependiendo de la decisión:
- Android nativo (Kotlin/Gradle): `app/src/main/java/...`, `app/src/main/res/...`, `build.gradle.kts`
- Flutter: `lib/`, `pubspec.yaml`
- Con backend propio: una carpeta hermana `backend/` con su propia estructura

No se asume ninguna de estas por adelantado — se define con la Arquitectura, no antes.
